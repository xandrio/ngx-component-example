ngx-component-example

ngx-component-example is a lightweight Angular wrapper component designed for showcasing other components in a configurable environment.
It provides built-in support for theme switching (default / dark) and text direction (LTR / RTL), making it perfect for component demos, style previews, and UI testing.

Usage:

To use the ngx-component-example component, simply import it into your Angular module and include it in your template. The component allows you to customize its appearance and behavior through various inputs and outputs.
